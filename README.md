---

## ğŸŒŸ Fitur Unggulan

### 1. Pencatatan Transaksi
- **Pemasukan & Pengeluaran**: Catat arus kas Anda dengan cepat.
- **Support Keterangan**: Tambahkan catatan untuk setiap transaksi.
- **Prediksi Saldo**: Tandai transaksi sebagai "akan datang" untuk melihat estimasi saldo di masa depan.
- **Offline First**: Awalnya tanpa internet? Tidak masalah. Data akan disinkronkan saat Anda kembali online (local storage).

### 2. Manajemen Alokasi (Dompet)
- **Multi-Akun**: Kelola saldo terpisah untuk Cash, Bank (BCA, SeaBank), E-Wallet (GoPay), dan lainnya.
- **Custom Alokasi**: Tambah atau hapus jenis dompet/alokasi sesuai kebutuhan Anda via menu Settings.
- **[BARU] Fitur Investasi**:
  - Catat pembelian investasi (Dompet -> Investasi).
  - Catat pencairan investasi (Investasi -> Dompet).
  - Monitoring saldo investasi terpisah.

### 3. Laporan & Analitik
- **Grafik Interaktif**: Visualisasikan tren pemasukan, pengeluaran, dan sisa saldo harian.
- **Ringkasan Saldo**: Lihat total aset dan sisa saldo per akun secara real-time.
- **Filter Canggih**: Filter berdasarkan tanggal, tipe transaksi, dan periode.

### 4. Kritik & Saran [BARU]
- **Feedback Langsung**: Kirim masukan, kritik, atau saran pengembangan langsung dari aplikasi.
- **Telegram Integration**: Pesan Anda akan terkirim otomatis ke developer.

### 5. Privasi & Keamanan
- **Mode Privasi**: Sembunyikan nominal saldo dengan satu klik (icon mata).
- **Lokal**: Data Anda milik Anda. Tidak ada data yang dikirim ke server pihak ketiga selain feedback yang Anda kirimkan sadar.

## ğŸ“± Cara Install (PWA)

Aplikasi ini dapat diinstal langsung ke HP Anda tanpa melalui Play Store (lebih ringan & hemat memori).

1. Buka [https://dhamarputra.github.io/JejakUang/](https://dhamarputra.github.io/JejakUang/) di **Google Chrome** (Android) atau **Safari** (iOS).
2. Tunggu prompt **"Add to Home Screen"** atau buka menu browser (titik tiga / tombol share).
3. Pilih **"Install App"** atau **"Tambah ke Layar Utama"**.
4. Aplikasi akan muncul di menu HP Anda dan bisa dijalankan layaknya aplikasi native beneran!
